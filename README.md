# AnimApi 
AnimApi is a public api built using node js


## Usage Api
1. Clone repository
```bash
git clone https://github.com/Ferdian9991/AnimApi.git
```
2. Install packages (using `yarn` or `npm`)
```bash
npm install
```
3. Start server
```bash
npm run start
```

## API Documentation
